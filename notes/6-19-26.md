# Thinking about workflow  
  1. You have mechanism tests, not phenotype tests. soma.rs/synapse.rs prove "alpha boosts by 64" and "beta increments by burst size." Nothing proves the
  neuron learns a pattern, that LTP/LTD have the right sign, or that weights don't run away. "Can this unit learn?" is currently unfalsifiable — which is
  exactly the thing you said you need to nail.
  2. The learning rule has no homeostasis. delta = (beta - H_BETA) * alpha / lr uses a fixed threshold H_BETA=4. Payeur's whole point (and BCM's) is that the
  threshold is an adaptive moving average P̄_j — that's what balances LTP vs LTD and keeps weights stable. With a constant, weights likely either saturate or
  die. This is your intelligence.tex BCM/Payeur section, not yet implemented.
  3. The interesting loop is open. The apical→soma teaching signal (APICAL_DV_S, the sigmoid) is marked "VALUES UNTUNED... calibrate once the feedback path
  runs." So right now you have feedforward Hebbian-ish updates with no top-down "confirmation" — which is precisely the L5 somatodendritic-residual algorithm
  that makes your model worth building. Without it, the model is generic, not Larkum.
  4. No inhibition. Weights init U(0,8), all-excitatory. Your tex is explicit that inhibition needs dedicated interneurons. Without it, lateral inhibition,
  winner-take-all, sparse codes, and rank-order coding (whole sections of your tex) literally cannot appear.
  5. Time is ambiguous. A "tick" = one event wavefront = one causal generation. But ALPHA_DECAY (halves/2048 ticks), T_BETA=500, eligibility traces (~5–10s in
  biology), STDP windows, and the 10ms-per-stage constraint all need real time (ms). Right now propagation depth and elapsed time are the same variable.
  Spike-timing and STDP phenomena can't be expressed faithfully on this substrate.

  | Phenotype | Source | Falsifiable assertion |
  | --------- | ------ | --------------------- |
  | Distal vs proximal EPSP gain | Branco & Hausser 2011 | distal EPSP ~95% across 1-10ms intervals; proximal drops to ~56% |
  | Dendritic NMDA spike | Major 2008/2013 | ~8 colocated synapses (close x) fire a branch; ~20 if spread |
  | Larkum 3 modes | Larkum model | FF regular spikes, FB burst, FF+FB sustained burst |
  | Burst dependent plasticity | Payeur 2021 | burst -> LTP of active synapses; non-burst event -> LTD; learning halts when post = target |
  | STDP | "Time dependent plasticity" | pre-before-post -> LTP, post-before-pre -> LTD |

  This is what turns "behaves in accordance with biological principles" from a vibe into a test suite.
  The capability ladder (start small — each rung gated by its phenotypes)

  1. Single synapse — EPSP shape; eligibility decay matches a target ms time-constant; LTP/LTD sign correct.
  2. Single dendrite — nonlinear ordered integration; NMDA spike from coincident colocated x; proximal vs distal gain.
  3. Single neuron (Larkum) — the 3 output modes; somatodendritic residual computed; burst plasticity stable (needs the sliding threshold).
  4. Pair + inhibition — introduce inhibitory neuron type; lateral inhibition; STDP between two cells; one neuron learns to predict another.
  5. Microcircuit — a small motif (your blobs place→hidden→output) learns a toy classification via rank-order or rate. First real "it learns" demo.
  6. Scale + GPU.
