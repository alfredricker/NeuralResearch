# High Level Contents
## Init
This is the directory with different initialization patterns.
* Neuron type initialization
* Circuit initialization
* Hyperparameters for dendrites, synapses, etc.

## Math
This directory contains important mathematical and statistical functions that are used in practice.

## Memory
This directory is for computer memory management. This is the code for determining network partitions that are loaded into the GPU.

## Neuron
This defines all the components necessary to create any type of neuron.

## Taxonomy
This is where you define specific neuron types such as Pyramidal L5, CA1, Visual Sensory, etc.
